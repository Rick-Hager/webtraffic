# WebTraffic

Site gerado automaticamente pela [Renovo](https://renovoweb.com.br).

- **Stack:** Astro static, hospedado na Vercel via integração Git nativa.
- **Deploy:** push em `main` → build + deploy automático.
- **Conteúdo:** página de placeholder em `src/pages/index.astro`.

Iterações futuras de conteúdo virão dos canônicos em `/clientes/webtraffic/`
(brand-book, site-content, etc.) — ver AGENTS.md §4.2.3 (Site Generator Agent).
